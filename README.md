## 说明
基于 [pikpakHelpr](https://github.com/xiaoziguys/pikpakHelpr) 改进的 Pikpak 增强脚本——调用官方 API 获取文件信息，支持勾选文件/文件夹一键推送到 Aria2 下载。

安装地址：https://greasyfork.org/zh-CN/scripts/503530-pikpak%E5%8A%A9%E6%89%8Bplus

加载脚本后，在 Pikpak 网页右侧会显示一个可拖拽的悬浮按钮，点击即可使用。

可用页面：

- 全部文件（`https://mypikpak.com/drive/all`，地址栏保持纯净）
- 任意文件夹页面
- 最近添加页面
- 分享页面（`/s/:shareId`）

## 代理相关

建议配置aria2走本地代理，否则可能没有速度

### 配置
安装后刷新在新建按钮旁边会出现一个aria2配置按钮，点击配置
- 服务器 `http(s)://host:port/jsonrpc`
- 路径 aria2的下载路径 `/downloads`
- 密钥 aria2的密钥
